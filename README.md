# PING-PONG-SCOREKEEPER
 
### Features
  1. KEEP TRACK OF PLAYER SCORE
  2. KEEP TRACK OF OPPONENT SCORE
  3. THREE BUTTONS TO INCREMENT AND RESET
  4. DISABLED BUTTON WHEN ONE PLAYER WINS
  5. ABLE TO SELECT THE WINNING SCORE
### Tech
  1. HTML
  2. CSS,BULMA
  3. JAVASCRIPT
